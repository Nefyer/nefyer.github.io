Why i started learning programming

Nov 9, 2024

Alright everyone, Chloe here, have decided to finally learn how to code. Yep, I’m diving straight into the wonderful world of programming – not just because it’s cooler than antartica, but because it’s actually super useful if you, like me, want to get into the universe (not literally, but close).

Now, before you ask, “Chloe, why put yourself through the existential dread of learning code?” let me tell you: I’m studying astronomy. And it turns out, if you’re into the astronomy, programming is pretty much mandatory. I’m talking data analysis, simulations, all the spicy stuff. And Python? It’s like the golden retriever of programming languages in astronomy stuff, everyone loves it, and honestly, it kinda carries the science world on its back.

So, here’s the plan: I’m giving myself three months to become a wizard with Python. Every day, me and Python, one-on-one, figuring out how not to crash my code every five seconds. Gonna build that foundation nice and sturdy, ‘cause once Python and I are best buds, I’m moving on to C++ you know, really doubling down on the nerd vibes. And yeah, the ultimate goal is to be good enough to start freelancing or maybe tutoring. I mean, why not make some cash while leveling up my skills?

For anyone thinking about learning Python, here’s a pro tip from me: start small. I know, everyone says that, but there’s a reason, treat it like grinding in a game. Level up a little bit every day, and before you know it, you’re a coding legend. Python is perfect for beginners, and it opens up a ton of doors.

Anyway, I’ll keep y’all posted on this journey, assuming I don’t end up huddled in the corner of my room, muttering about syntax errors. But hey, every crash brings me one step closer to that Daigaku I've been dreaming for so long. So buckle up, this is gonna be one wild coding ride.
